# Heart Disease Risk Prediction

## Overview

This project predicts heart disease risk using machine learning models trained on health indicator data.

## Dataset

Heart Disease Health Indicators Dataset

Features include:

- BMI
- Smoking
- Diabetes
- High Blood Pressure
- Physical Activity
- Age
- Sex
- Income

Target:

- HeartDiseaseorAttack

## Models Used

1. Logistic Regression
2. Random Forest
3. XGBoost

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Explainable AI

SHAP was used to identify the most influential features contributing to heart disease risk.

## Results

Model comparison results are available in:

- results/model_performance.csv

## Key Findings

- High Blood Pressure is a major predictor.
- Age significantly impacts risk.
- BMI contributes strongly.
- Diabetes and Smoking increase risk.
